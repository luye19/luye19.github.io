基于心电信号的心血管疾病智能辅助诊断系统

Mar 27, 2024 452
项目简介

本研究提出了一个通道和时序特征融合的新型的双分支注意力融合的卷积神经网络模型用于 12 导联心电图的自动诊断。具体来说，由于心电信号的时间维度复杂，本研究引入一维卷积（Convolutional neural network，CNN）模块来提取低维特征，降低计算复杂度。但是，由于CNN的感受野范围被卷积核大小限制，其善于提取局部细粒度特征，却难以对特征间的全局关系进行挖掘。针对十二导联心电图的多导联和多时间点的特点，本研究设计了双分支Transformer，两个分支分别提取心电信号的通道特征和时序特征，从而能深度挖掘到心电图信号所蕴含深层次病理特征。

项目成果
  1. 论文：Gated CNN-Transformer Network for Automatic Cardiovascular Diagnosis using 12-lead Electrocardiogram （EMBC，口头汇报，一作）
  2. 专利：一种心电图分类方法、分类装置、设备及存储介质（实审中）
  3. 第二届广东省生物医学工程创新设计竟赛二等奖
